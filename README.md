# Customer Segmentation Dashboard (Machine Learning Project)

## Project Overview

- This project uses **Machine Learning (K-Means Clustering)** to analyze customer purchasing behavior and divide customers into different segments based on their income and spending patterns.
- The system provides an **interactive dashboard** where users can visualize customer clusters and understand different types of customers.
- The goal of this project is to help businesses understand their customers better and create **data-driven marketing strategies**.

# Features

* Data preprocessing of customer dataset
* Feature scaling for better clustering
* Elbow method to determine optimal number of clusters
* PCA visualization of clusters
* Interactive dashboard built with Flask
* Dataset visualization page
* Customer segment statistics

# Technologies Used

* Python
* Flask
* Pandas
* Scikit-learn
* Plotly / Matplotlib
* HTML
* CSS

# Dataset

Dataset used: **Mall Customers Dataset**

Features in dataset:

* CustomerID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1–100)


# How to Run the Project

## 1. Clone the Repository

git clone https://github.com/your-username/customer-segmentation-dashboard.git

Go to the project folder : cd customer-segmentation-dashboard


## 2. Install Required Libraries

Install dependencies using pip

pip install flask pandas scikit-learn matplotlib plotly

## 3. Run the Application

python app.py

## 4. Open in Browser

Open the following URL:

http://127.0.0.1:5000
